# Environment Setup

## Installations

* Install `miniconda`
    ```shell
    brew install miniconda
    ```
* Copy the below command to shell profile
    ```text
    eval "$(conda "shell.$(basename "${SHELL}")" hook)"
    ```
* Clone the repo from terminal
    ```shell
    git clone https://github.com/sushovan/tinygpt.git  
    ```
* Open pycharm.
    ```shell
    pycharm .
    ```
   It will automatically prompt to create a conda environment.
